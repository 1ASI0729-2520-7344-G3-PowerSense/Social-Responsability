# Fundamentos de Java para Principiantes

## Resumen del Curso

Este curso de **1 hora** introduce a estudiantes de secundaria a los **fundamentos de la programación con Java**, incluyendo conceptos básicos de **programación orientada a objetos (POO)**.  
**¡No requiere descargas!** Solo abre tu navegador web.

- **Duración total**: ~60 minutos  
- **Público objetivo**: Estudiantes de 12 a 17 años sin experiencia en programación  
- **Prerrequisitos**: Ninguno  
- **Herramientas necesarias**: **Solo tu navegador web** (Chrome, Firefox, Safari, Edge)  

> Todo el código se ejecuta en editores online gratuitos como Replit y JDoodle. No es necesario crear cuentas para comenzar a programar.

**Repositorio de Código Fuente**:  
[Repositorio de GitHub – Fundamentos de Java](https://github.com/1ASI0729-2520-7344-G3-PowerSense/Social-Responsability)

  - Un **array de objetos** permite manejar varias instancias, por ejemplo `Mascota[]`.
  - Un programa POO básico combina clases, objetos, métodos, condiciones y bucles.
