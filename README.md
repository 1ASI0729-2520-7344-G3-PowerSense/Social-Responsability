# Fundamentos de Java para Principiantes

## Resumen del Curso

Este curso de **1 hora** introduce a estudiantes de secundaria a los **fundamentos de la programación con Java**, incluyendo conceptos básicos de **programación orientada a objetos (POO)**.  
**¡No requiere descargas!** Solo abre tu navegador web.

- **Duración total**: ~60 minutos  
- **Público objetivo**: Estudiantes de 12 a 17 años sin experiencia en programación  
- **Prerrequisitos**: Ninguno  
- **Herramientas necesarias**: **Solo tu navegador web** (Chrome, Firefox, Safari, Edge)  

> Todo el código se ejecuta en editores online gratuitos como Replit, JDoodle u OnlineGDB. No es necesario crear cuentas para comenzar a programar.

## Secuencia de la Lección

### Lección 1: ¿Qué es programar? Introducción a Java y al editor online (5 minutos)

- **Descripción**:  
  Conoce qué es la programación, para qué sirve Java y por qué es un buen lenguaje para empezar. Se presenta el editor de código online y se ejecuta el clásico “Hola Mundo”.

- **Enlace**:  
  [Ver la lección]()

- **Conclusiones clave**:
  - Un programa es un conjunto de instrucciones que la computadora sigue paso a paso.
  - Java se usa en aplicaciones móviles, juegos, sitios web y sistemas grandes.
  - Un programa simple en Java tiene una clase y un método `main`.

- **Empieza a programar**:  
  [Abrir Replit – Hola Mundo en Java]()  
  > El estudiante modifica el mensaje en pantalla cambiando el texto de `System.out.println`.

---

### Lección 2: Variables, tipos de datos y operadores (10 minutos)

- **Descripción**:  
  Aprende a **guardar información** con variables, usar tipos de datos básicos (`int`, `double`, `String`, `boolean`) y aplicar operadores aritméticos y de comparación.

- **Enlace**:  
  [Ver la lección]()

- **Conclusiones clave**:
  - Una variable es “una caja” donde se guarda un valor.
  - Tipos básicos: `int` (números enteros), `double` (decimales), `String` (texto), `boolean` (verdadero/falso).
  - Operadores comunes: `+`, `-`, `*`, `/`, `%`, `>`, `<`, `==`.

- **Práctica**:  
  [Clic para programar – Calculadora sencilla]()  
  > El estudiante declara variables para edad, nota o puntaje, y calcula resultados simples.

---

### Lección 3: Estructuras de control – Decisiones y bucles (10 minutos)

- **Descripción**:  
  Toma decisiones con `if-else` y repite instrucciones con bucles `for` y `while`. Se construyen ejemplos cercanos al estudiante (por ejemplo, verificar si aprueba un curso o contar intentos en un juego).

- **Enlace**:  
  [Ver la lección]()

- **Conclusiones clave**:
  - `if (condición) { ... } else { ... }` permite que el programa “decida”.
  - Un bucle `for` repite un bloque un número conocido de veces.
  - Un bucle `while` repite mientras la condición sea verdadera.

- **Práctica**:  
  [Clic para programar – Contador de intentos]()  
  > El estudiante crea un bucle que muestra los intentos del 1 al 5 y usa un `if` para mostrar un mensaje especial en el último intento.

---

### Lección 4: Métodos, entrada de datos y primeros pasos hacia la POO (12 minutos)

- **Descripción**:  
  Aprende a organizar mejor el código usando **métodos** (funciones) y a leer datos desde la consola con `Scanner`. Se introduce la idea de que un programa se puede dividir en “acciones” reutilizables.

- **Enlace**:  
  [Ver la lección]()

- **Conclusiones clave**:
  - Un método es un bloque de código reutilizable, por ejemplo:  
    `public static void saludar() { ... }`
  - `Scanner` permite leer información que el usuario escribe en la consola.
  - Separar el código en métodos hace el programa más ordenado y fácil de entender.

- **Práctica**:  
  [Haz clic para practicar – Programa que saluda al usuario]()  
  > El estudiante crea un método `saludarUsuario` que pida el nombre por consola y muestre un mensaje personalizado.

---

### Lección 5: Introducción a la POO – Clases, objetos y constructores (13 minutos)

- **Descripción**:  
  Se introduce formalmente la **programación orientada a objetos (POO)**: clases, objetos, atributos y constructores. Se construye una clase simple, por ejemplo `Estudiante`, con algunos datos y comportamientos básicos.

- **Enlace**:  
  [Ver la lección]()

- **Conclusiones clave**:
  - Una **clase** es el “molde” (plantilla); un **objeto** es un “ejemplar” de esa clase.
  - Los **atributos** guardan información (por ejemplo, `nombre`, `edad`).
  - El **constructor** sirve para inicializar un objeto cuando se crea con `new`.

- **Práctica**:  
  [Clic para programar – Clase Estudiante]()  
  > El estudiante crea la clase `Estudiante` con atributos `nombre` y `edad`, un constructor y un método que muestre la información del estudiante.

---

### Lección 6: Encapsulación, arreglos y mini proyecto “Registro de Mascotas” (10 minutos)

- **Descripción**:  
  Se presenta la **encapsulación** usando getters y setters, se usa un **array** de objetos y se construye un mini proyecto: un **registro simple de mascotas** en consola.

- **Enlace**:  
  [Ver la lección]()

- **Conclusiones clave**:
  - Encapsulación: se usan `private` para los atributos y métodos `get` / `set` para acceder a ellos.
  - Un **array de objetos** permite manejar varias instancias, por ejemplo `Mascota[]`.
  - Un programa POO básico combina clases, objetos, métodos, condiciones y bucles.

- **Proyecto final – Registro de Mascotas**:  
  [Crea tu aplicación POO]()  
  > El estudiante construye una clase `Mascota` (nombre, tipo) y un pequeño menú en consola para registrar y mostrar varias mascotas.
